ExcelAssistAI

AI-Powered Chatbot for College Admission Support

ExcelAssistAI is an agentive, Generative AI-based chatbot built to streamline the college admission process for Excel Engineering College. It serves as a virtual counselor, providing real-time assistance to students and parents seeking information about admissions, courses, fees, and campus facilities. Developed using Python, Gradio, and Google’s Gemini API, the chatbot is fully functional, intelligent, and publicly accessible on Hugging Face Spaces.

  • Conversational Assistant: Seamlessly handles a variety of queries, including department offerings, application deadlines, eligibility criteria, scholarship info, hostel facilities, and placement records.
  • Natural Language Understanding: Powered by Google’s Gemini Pro, it delivers context-aware, precise, and human-like responses using advanced prompt engineering techniques.
  • Agentive AI Features: Not only responds to user queries but proactively suggests relevant information (e.g., deadlines, recommended courses) based on the flow of conversation.
  • Prompt Engineering: Carefully structured prompts are used to guide the model in maintaining tone, relevance, and factual accuracy specific to the college domain.
  • Deployment: Hosted on Hugging Face Spaces for global access, no login required.
  • User Interface: Built with Gradio, the UI is clean, responsive, and optimized for both mobile and desktop, ensuring accessibility to users with minimal technical background.
  • Tech Stack: Python, Gradio, Gemini API, Hugging Face Spaces, Prompt Engineering, Agentive AI architecture.
  • Scalability: Designed with future integration in mind — can be extended to handle fee payment queries, document uploads, or even live chat with college staff via API bridge.

College Shuttles Tracking App

Real-Time Vehicle Tracking System for Campus Commute

X-Link is a real-time shuttle tracking web application developed to improve transportation efficiency and safety across the college campus. Designed for students, faculty, and administrative staff, the app provides accurate shuttle locations, estimated arrival times, and live route visualizations. Built with a responsive interface and powered by GPS data, X-Link enhances daily commuting within the college premises.

  • Live Location Tracking: Utilizes GPS data to fetch and display real-time shuttle positions on an interactive map using tools like Leaflet or Google Maps API.
  • Estimated Arrival Time (ETA): Dynamic ETA updates based on vehicle speed and traffic conditions to help users plan their commute.
  • Multi-Role System: Designed with separate interfaces for Students, Admins, and Drivers, ensuring tailored access and functionality.
  • Admin Dashboard: Admins can manage shuttle routes, track vehicle status, edit schedules, and send out notifications for delays or maintenance.
  • Notifications: Push alerts for shuttle arrivals, delays, or route changes via in-app messages or SMS/email integration (optional).
  • Responsive UI: Built with mobile-first design principles using HTML5, CSS3, Bootstrap, and JavaScript (React/Vanilla JS), ensuring cross-device usability.
  • Tech Stack: MERN Stack (MongoDB, Express.js, React.js, Node.js), Map APIs, and optional Firebase/Socket.IO for real-time updates.
  • Security & Auth: Includes JWT-based authentication and role-based access control to protect user and admin operations.
  • Deployment: Can be hosted on platforms like Netlify (Frontend) and Render/Heroku (Backend), with MongoDB Atlas as cloud DB.
  • Impact: Minimizes wait times, reduces congestion at pick-up points, and improves shuttle utilization across campus.
  • GitHub: X-Link Repository
